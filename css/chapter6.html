flex

フレックスボックス レイアウトは、CSS を使用して Web ページをより柔軟かつ効率的にデザインする方法です。さまざまな画面サイズに合わせて応答性の高い動的なレイアウトを作成するのに役立ちます。

ここにフレックスボックスのプロパティがあります

  • display
  • flex-direction
  • justify-content: 水平軸を定義します。
  • align-items: 縦軸を定義します。
  • flex

display プロパティ

`display: flex;` プロパティを使用して、フレックスボックス コンテナを定義できます。

例: 単純なフレックスボックス レイアウトの作成

わかりやすくするために各要素の背景色とマージンを設定します。

css 
<!DOCTYPE>
<html>
<head>
<title >LearnFreely</title>
<style>
  .display {
    display: flex;
  } 

  p {
    background-color: green;
    margin: 4px;
  }
</style>
</head>
<body>

  <div class="dispaly" >
    <p>これは通常の段落です。</p>
    <p>この要素は block 要素です。</p>
    <p>もう一つの通常の段落 </p>
  </div>

</body> 
</html>
  • 「display:flex;」: コンテナをフレックスコンテナに変えます。
  • この < div > 要素の直接の子はすべて flex アイテムになります。

フレックスボックス レイアウト結果：

これは通常の段落です。

この要素は block 要素です。

もう一つの通常の段落.
