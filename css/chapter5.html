Float and Claer

Float

Float は、要素を通常のフローから取り出して、その要素を含むボックスの左側または右側に配置できるようにする CSS プロパティです。

構文

float: 値;

値

1. `float: left;`
この値は要素を左側にフロートさせ、他の要素のコンテンツを右側に表示できるようにします。
2. `float: right;`
この値は要素を右側にフロートさせ、他の要素のコンテンツを左側に表示できるようにします。
3. `float: none;`
これがデフォルト値です。フロートなし

フローティング要素はページ上のコンテンツのフローに影響を与える可能性があるため、「float」プロパティを使用する場合は、他の要素のレイアウトと位置を考慮することが重要です。さらに、フローティング要素の後の要素の動作を管理するには、「clear」プロパティの使用が必要になることがよくあります。

Clear

Clear は、要素の片側または両側に浮動(フローティング)要素が許可されないことを保証する CSS プロパティです。

構文

clear: 値;

値

1. `clear: left;`
この値により、要素は左浮動要素の下に確実に移動されます。
2. `clear: right;`
この値により、要素は右浮動要素の下に確実に移動されます。
3. `clear: both;`
この値により、要素は左または右の浮動要素の下に移動されます。両面クリアです。
3. `clear: none;`
これがデフォルト値です。

例を見てみましょう。

css 
<!DOCTYPE>
<html>
<head>
<title >LearnFreely</title>
<style>
  .float_rei {
    float: left;
  } 
</head>
<body>
<p>これは通常の段落です。</p>
<p> class="float_rei">この要素は block 要素です。</p>>
<p>もう一つの通常の段落 </p>
</body> 
</html>
  • この例では、要素を左にフローティングします。

結果：

これは通常の段落です。

この要素は block 要素です。

もう一つの通常の段落.

右側のコンテンツを新しい行で開始したい場合は、claer プロパティを使用する必要がある。

clear プロパティを適用してみましょう。

css 
<!DOCTYPE>
<html>
<head>
<title >LearnFreely</title>
<style>
  .float_left {
    float: left;
  } 
  
  .clear_left {
    claer: left;
  } 
</head>
<body>
<p>これは通常の段落です。</p>
<p class="float_left">この要素は block 要素です。</p>>
<p class="clear_left">もう一つの通常の段落 </p>
</body> 
</html>

これは通常の段落です。

この要素は block 要素です。

もう一つの通常の段落.
