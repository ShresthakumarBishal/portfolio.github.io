CSSの基本

HTML に CSS を追加する

CSS (Cascading Style Sheets) は、HTML ドキュメントのスタイルを設定するために使用される言語です。 このセクションでは、HTML ファイルに CSS を組み込むさまざまな方法を学習します。

以下に示すように、さまざまな方法で CSS を HTML ファイルに組み込む方法を学習します。

  1. インラインCSS
  2. 内部CSS
  3. 外部CSS

2. インラインCSS

まずは、HTML タグ内で直接 style 属性を使用する、最も単純な方法から始めます。

例えば：

<p class="color: blue; font-size: 16px;">インラインCSS</p>

内部CSS

次に、ドキュメントの <html> の <style> タグ内に CSS が含まれていることを確認します。 このメソッドを使用すると、複数の要素のスタイルを設定でき、スタイルを HTML 構造から切り離して維持できます。

  <!DOCTYPE>
  <html>
  <head>
    <style>
      <p> {
        color: green;
        font-size: 18px;
      }          
    </style>      
  </head>
  <body>
    <p >インラインCSS</p>
  </body> 
  </html>

3. 外部CSS

最後に、CSS を外部ファイル (styles.css など) に配置し、HTML にリンクすることの利点を理解します。 これにより、保守性と再利用性が向上します。

例えば:

<!DOCTYPE>
<html>
<head>
  <p rel=stylesheet type="text/css"  href="style.css">
</head>
<body>
  <p >インラインCSS</p>
</body> 
</html>

style.css

p {
  color: green;
  font-size: 18px;
}

CSSセレクター

CSS セレクターは、HTML 要素のスタイルを設定するために使用されます。 セレクターの基本的な種類とその使用方法を学習します。

要素セレクター

特定の HTML 要素のすべてのインスタンスをスタイルします。

css 
  p {
    color: green;
    font-size: 18px;
  }

クラス セレクター

同じクラス属性値を持つすべて要素を対象とします。

css 
  .greenBox {
    color: green;
    font-size: 18px;
  }

IDセレクター

同じID属性値を持つ要素を対象とします。

css 
  #title {
    color: green;
    font-size: 18px;
  }

Web ページ上のテキストのスタイルと書式設定を可能にするさまざまな CSS プロパティを見ていきます。

テキストのプロパティ

Color

テキストの色を指定します。 色名、16 進値、RGB、または HSL 値を使用して定義できます。

例えば：

css 
.greenBox {
  color: green;
  font-size: 18px;
}

text-align

コンテナ内のテキストを位置合わせします。 オプションには、左、右、中央、および両端揃えが含まれます。

例えば：

css 
.greenBox {
  text-align: center;
}

行の高さ

テキスト行の高さを設定します。 行の高さを適切にすると、読みやすくなります。

例えば：

css 
    .greenBox {
      line-height: 1.5;
    }

行の高さ

テキスト行の高さを設定します。 行の高さを適切にすると、読みやすくなります。

例えば：

css 
    .greenBox {
      line-height: 1.5;
    }

font-sizer

テキストのサイズを決定します。 px、em、rem、%などのさまざまな単位を使用できます。

例えば：

css 
.greenBox {
  font-size: 18px;
}

font-family

このプロパティはテキストのフォントを定義します。 特定のフォント、または「sans-serif」や「serif」などの汎用フォント ファミリを指定できます。

例えば：

css 
.greenBox {
  font-family: 'Open Sans', Arial, sans-serif;;
}

font-weight

フォントの太さを設定します。 一般的な値には、normal、bold、bolder、lighter などがあります。

例えば：

css 
.greenBox {
  font-family: 'Open Sans', Arial, sans-serif;;
}

Styling Links

text-decoration

リンクに使用される装飾を制御します。 値には、下線、上線、中抜き線、およびなしが含まれます。

例えば：

css 
  a {
    text-decoration: none;
  }

background-color

CSS プロパティを使用して背景を設定する方法を学びます。 ここでは、background-color、background-image、background-repeat、background-position、background-size について説明します。

css 
  div {
    background-color: #f2f2f2;
  }
css 
  div {
    background-image: url('background-image.jpg');
    background-repeat: no-repeat;
    background-position: cover;
  }

Transparency and Opacity

RGBA および HSLA カラー値を使用して要素を透明にする方法を学びます。 opacity プロパティを使用して要素全体の不透明度を制御する方法を学びます。

css 
   /* Adding transparency with RGBA */
  div {
    background-color: rgba(255, 0, 0, 0.5);
  }   
  
  /* Using HSLA for transparency */
  div {
    background-color: hsla(120, 100%, 50%, 0.3);
  }  
  
  /* Controlling element opacity */
  div {
    opacity: 0.8;
  }

CSS での色の表現と背景の操作についてしっかりと理解できるようになります。
