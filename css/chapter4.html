Display

表示プロパティは、要素のレンダリングを決定します。さまざまな表示プロパティ値とその適切な使用法について説明します。

構文:

display: 値;

display プロパティには、HTML要素の表示方法を定義するさまざまな値があります。

共通の値

  1. block (ブロック表示)
  2. inline (インライン表示)
  3. inline-block (インラインブロック表示)
  4. none(なし)

block

CSS プロパティ「display: block;」 HTML 要素をブロックレベルの要素のように動作させるために使用されます。以下に例を示しながら段階的に説明します。

覚えておくべきポイント

  • ブロックレベルの要素としてレンダリングします。
  • 全幅を使用します。
  • 新しい行で始まります。

例：

css 
<!DOCTYPE>
<html>
<head>
  <title >LearnFreely</title>
  <style>
      .block_rei {
        display: block;
      } 
</head>
<body>
  <p>これは通常の段落です。</p>
  <span class="block_rei">この要素は block 要素です。</span>
  <p>もう一つの通常の段落 </p>
</body> 
</html>
  • 「この要素は block 要素です。」テキストは全幅を占めるブロックとして表示されます
  • 「display:block;」により新しい行から始まります。

結果：

これは通常の段落です。

この要素は block 要素です。

もう一つの通常の段落.

inline

CSS の「display: inline」プロパティは、要素をインライン要素のように動作させるために使用されます。以下に例を示しながら段階的に説明します。

覚えておくべきポイント

  • インラインレベルの要素としてレンダリングします。
  • 必要な幅だけを取ります。
  • 新しい行では始まりません。

2 つの HTML 要素 (段落

など) があると仮定します。

例：

css 
<!DOCTYPE>
<html>
<head>
  <title >LearnFreely</title>
  <style>
      p {
        display: block;
      } 
</head>
<body>
  <p>これは通常の段落です。</p>
  <p>もう一つの通常の段落 </p>
</body> 
</html>
  • 段落はインラインで表示されます。
  • 新しい行で始まることはなく、その幅は必要なだけです。
  • 視覚的にわかりやすくするために、背景色が追加されています。

「display: inline」を適用した場合の結果

タイトル2

「display: inline」を適用しない場合の結果

これは通常の段落です。

もう一つの通常の段落.

  • p 要素はブロック要素であり、全幅をとり、新しい行で始まります。

これは、Web ページ上の要素をよりコンパクトでインラインに配置する場合に特に役立ちます。

inline-block

CSS の「display: inline-block」プロパティは、要素をインライン レベルのブロック コンテナーにするために使用されます。

覚えておくべきポイント

  • ブロック要素をインラインブロック要素にするときに利用します。/li>

2 つの HTML 要素 (段落

など) があると仮定します。

例：

css 
<!DOCTYPE>
<html>
<head>
<title >LearnFreely</title>
<style>
    p {
      display: inline-block;
      display: 200px;
      display: 150px;
    } 
</head>
<body>
<p>これは通常の段落です。</p>
<p>もう一つの通常の段落 </p>
</body> 
</html>
  • 「display: inline-block;」 それをインラインブロック要素にします。
  • 「width: 200px;, height: 150px;」 ボックスの幅と高さを設定します。
