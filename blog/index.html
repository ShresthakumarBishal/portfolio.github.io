Simple Kivy App

To create a Simple Kivy App with Python, we will write some code in a main.py file, which is a Python script that holds the main logic of our program. The code given below is an example of how to create a basic Kivy App that displays a button and a label. Open IDE, create a main.py file, and then write code similar to the one given below.

KV Language

KV language is a design language that is used to make the GUI/Layout of a kivy Application. Now, We will make some buttons using the KV language. Kivy 2.2.1 documentation: This is the official documentation of Kivy
Delete main.py file, and then create a two files as given below:

  1. firstApp.py: For the main logic of our program.
  2. layout.kv: For the GUI/Layout design.

  // 1.firstApp.py
  from kivy.app import App
  from kivy.uix.screenmanager import ScreenManager, Screen  
  from kivy.uix.button import Button

  from kivy.lang import Builder

Screen Switch

After making two buttons and one label with the kv language, we will switch the screen by pressing button 1 and change the home() class label text by pressing button 2.

Let's add a few codes in the firstApp.py file to create another screen.
  // firstApp.py
  from kivy.app import App
  from kivy.uix.screenmanager import ScreenManager, Screen
  from kivy.uix.button import Button
  # import kv.kv file
  from kivy.lang import Builder
  Builder.load_file("kv.kv")
  
  #Screen layout class
  class Home(Screen):
      def changeText(self): #self represents Home() class
          self.ids.id_label.text="You pressed button 2"
          # self.ids means ids of Home() class from kv.kv
          # id_label is the id of Label from kv.kv
          # text is the text of Label
          
  #Another Screen layout class
  class AnotherScreen(Screen):
      pass
  
  class FirstKivy(App):
      def build(self):
          # Create the screen manager
          sm = ScreenManager()
          sm.add_widget(Home(name='FirstScreen'))
          sm.add_widget(AnotherScreen(name='SecondScreen'))
          return sm
  
  if __name__ == '__main__':
      FirstKivy().run()

  // quiz.py
  from kivy.app import App
  from kivy.uix.screenmanager import ScreenManager, Screen
  from kivy.properties import StringProperty, NumericProperty, ObjectProperty, ListProperty
  from kivy.core.window import Window
  import csv
  # import kv.kv file
  from kivy.lang import Builder
  Builder.load_file("quiz.kv")

  #Home Screen layout class
  class Home(Screen):
      pass

  #Quiz Screen layout class
  class QuizScreen(Screen):
      # set variables
      question= StringProperty()
      ans= StringProperty()
      ans1= StringProperty()
      ans2= StringProperty()
      Ans_data= StringProperty()
      error= StringProperty()
      error_number= NumericProperty(0)
      page_number=NumericProperty()
      no=NumericProperty()
      df=ListProperty()
      correct_answer=StringProperty()

      def __init__(self, **kwargs):
          super(QuizScreen, self).__init__(**kwargs)
          self.no=int(1)
          self.page_number= int(1)
          op=open('question.csv', 'r', encoding='utf-8')
          self.df=list(csv.reader(op))
          self.question=self.df[self.no][0]
          self.correct_answer=self.df[self.no][1]
          self.ans=self.df[self.no][2] #Answer for option Button 1
          self.ans1=self.df[self.no][3] #Answer for option Button 2
          self.ans2=self.df[self.no][4] #Answer for option Button 3

      def restart(self):
          self.no= int(0) # Restart from Q1
          self.error_number=int(0) # set 0 wrong
          self.page_number=int(0) # Set 1 page
          self.go_next(self.correct_answer) # call go_next() functon

      def go_next(self, find):
          self.error ='' #Set error variable to NON

          if  self.correct_answer == find:
              self.no=self.no+1 #for next question
              self.page_number=self.page_number + 1
              self.question=self.df[self.no][0]
              self.correct_answer=self.df[self.no][1]
              self.ans=self.df[self.no][2]
              self.ans1=self.df[self.no][3]
              self.ans2=self.df[self.no][4]                  
          else:
              self.error='[color=#FF0000]Wrong Answer Try Again[/color]'
              self.error_number=self.error_number+1 
      
  class QuizApp(App):
      def __init__(self, **kwargs):
          super(QuizApp, self).__init__(**kwargs)
          # Background color of app
          Window.clearcolor = (1,1,1,1) 
          #Size of app
          Window.size = (330, 600)

      def build(self):
          # Create the screen manager
          sm = ScreenManager()
          sm.add_widget(Home(name='HomeScreen'))
          sm.add_widget(QuizScreen(name='QuizScreen'))
          return sm

  if __name__ == '__main__':
      FirstKivy().run()

Deploy the App

For Window

For Windows, you can use PyInstaller to create a standalone executable file that contains your app and all the dependencies. PyInstaller is a tool that analyzes your code and bundles it with the required libraries and modules. You can also customize the appearance and behavior of your app using PyInstaller‚Äôs options and hooks. You can find more details on how to use PyInstaller for Windows. Click Here

For Android

For Android, you can use Buildozer or python-for-android to create an APK or AAB file that can be installed on your device or published on the Google Play Store. Buildozer is a tool that automates the entire build process by downloading and setting up all the prerequisites for python-for-android, including the Android SDK and NDK. Python-for-android is a tool that allows you to package your Python code and native libraries into a distributable format for Android. You can also use the Kivy Launcher app to run your Kivy app without compiling it, but this method has some limitations. You can find more details on how to use Buildozer, python-for-android, and Kivy Launcher for Android.Click Here

For MacS

For MacOS, you can use PyInstaller, Buildozer, or Homebrew to create a standalone app bundle that can be distributed or installed on your device. PyInstaller and Buildozer work similarly as for Windows and Android, respectively. Homebrew is a package manager that allows you to install Python and other dependencies on your system. You can find more details on how to use PyInstaller, Buildozer, and Homebrew for macOS. Click Here

For IOS

For iOS, you can use python-for-ios to create an Xcode project that can be compiled and deployed to your device or published on the App Store. Python-for-ios is a tool that allows you to package your Python code and native libraries into a distributable format for iOS. You can also customize the Xcode project using various options and settings. You can find more details on how to use python-for-ios for iOS. Click Here
